# OpenGallery WebApp

> Fast, open-source, self-hosted alternative to Google Photos.


## Getting Started

To get started with the web app, follow these steps:

1. npm install
2. npm run dev
3. Open your browser and navigate to `http://localhost:4200` to view the app.



## Technology Stack
- **Framework**: [Angular](https://angular.io/)
- https://tanstack.com/query/v5/docs/framework/angular/devtools
